# ✈️ Jornada Viagens

O **Jornada Viagens** é um projeto de agência de viagens desenvolvido para oferecer uma experiência de usuário moderna e totalmente responsiva. Este projeto faz parte do meu portfólio como estudante de **Engenharia de Software na FIAP**.

## 🚀 Tecnologias Utilizadas

- **Frontend:** HTML E CSS
- **Estilização:** CSS3 (Flexbox, Media Queries e Variáveis)
- **Deploy:** GitHub Pages com GitHub Actions

## 🎨 Diferenciais do Projeto

- **Design Adaptável:** Layout corrigido para ocupar 100% da largura da tela, eliminando barras brancas laterais indesejadas.
- **UI/UX:** Foco em hierarquia visual, tipografia Montserrat e acessibilidade.
- **Mobile First:** Interface pensada primeiro para dispositivos móveis e expandida para desktops de 1440px.
